### 2️⃣ Questions কোথায় Create করবে:

**Dashboard → Cost Management → Cost Codes tab**

#### Step 1: Parent Question তৈরি করো
```
1. "Add Cost Code" button click করো
2. Normal ভাবে fill করো:
   - Name: "Wall tile removal?"
   - Question Type: BLUE
   - Price: 0
3. Nested Question Settings section এ:
   - Parent Question: "None" রাখো
4. Save করো
```

#### Step 2: Child Question তৈরি করো
```
1. আবার "Add Cost Code" click করো
2. Fill করো:
   - Name: "Enter sqft of tile"
   - Question Type: GREEN
   - Price: 5.50
3. Nested Question Settings section এ:
   - Parent Question: "Wall tile removal?" select করো
   - Show When Parent Value: "true" লিখো
   - Nested Input Type: "QUANTITY" select করো
4. Save করো
```
