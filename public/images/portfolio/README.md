# Portfolio Images

Add project images to this folder with the following naming convention:

## Lakeview Project 1:
- `lakeview-1.jpg` to `lakeview-12.jpg`

## Lakeview Project 2:
- `lakeview2-1.jpg` to `lakeview2-8.jpg`

## Image Guidelines:
- Format: JPG or PNG
- Recommended size: 800-1200px width
- Optimize images for web (compress before uploading)
- Use descriptive names for easy management

The masonry grid will automatically adjust to different image heights.
